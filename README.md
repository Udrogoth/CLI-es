

Esta es la versión estable del intérprete para el lenguaje .es, compilado y listo para ejecutar en entornos Windows.

Cómo usarlo
Descarga el archivo CLI-es-win.exe adjunto.

**Full Changelog**: (https://github.com/Udrogoth/CLI-es/releases/tag/CLI-es)

Abre tu terminal en la carpeta donde lo guardaste.

Ejecuta cualquier archivo **.es, con la sintaxis correspondiente** con el comando:

**.\CLI-es-win**    _nombre_de_tu_archivo.es_
